# Projekti i Grupit

Ky është një projekt i përbashkët i studentëve.  
Përmban kod në C++ dhe të dhëna të organizuara në foldera të ndryshëm.

## Struktura
- `Januz/` → përmban të dhënat e përfunduara
- `Yll/` → përmban detyrat në C++
- `Rinion/` → përmban funksionin për gjetjen e notës mesatare
- `Ledion/` → përmban pjesën e fundit të projektit
